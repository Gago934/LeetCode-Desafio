# LeetCode-Desafio
## Binary Search

O desafio era criar um código de pesquisa binária, reforçando a lógica e a prática de algoritmos.
Com esse exercício, pude treinar o raciocínio passo a passo, entender como dividir problemas e aplicar conceitos do livro "Entendendo algoritmos" de forma prática e dinâmica.
